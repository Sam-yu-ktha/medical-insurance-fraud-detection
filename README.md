# Medical Insurance Fraud Detection

## 📌 Project Overview
This project focuses on detecting fraudulent medical insurance claims using machine learning techniques.

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib / Seaborn
- Google Colab

## 📊 Dataset
The dataset contains insurance claim details including:
- Patient information
- Claim amount
- Diagnosis
- Treatment details
- Fraud label

## 🚀 Model Used
- Logistic Regression
- Random Forest
- Decision Tree

## 🎯 Goal
To build a predictive model that can classify insurance claims as fraudulent or legitimate.
